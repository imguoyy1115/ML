COVID19 Regression Prediction - 李宏毅深度学习作业

项目简介：本项目为李宏毅深度学习课程课后回归作业1，任务是利用全连接神经网络，根据多维度时序特征预测新冠确诊人数。数据集呈现指数爆炸式增长特点，通过多次模型激活函数调试、优化器选型、数据分布变换、正则化与学习率调度调优，最终拟合效果与泛化误差达到优秀水平。

项目难点与核心思路：新冠确诊人数随时间呈指数级爆发增长，原始数据分布非线性极强，普通神经网络直接拟合难以收敛、误差极大，且极易出现过拟合；因此本项目核心思路为通过对数变换将指数增长转为近似线性分布，再配合网络结构、激活函数、优化器、正则化、学习率调度全方位调参，完成高精度回归预测。

模型结构：三层全连接回归网络
Linear → ReLU → Dropout
→ Linear → ReLU → Dropout
→ Linear → ReLU → Dropout
→ Linear(输出1维预测值)

调试与改进全过程：
1. 激活函数迭代调试：初期先使用Sigmoid函数搭建全连接网络发现训练效果差、验证集Loss很高，拟合能力不足：随后尝试更换为 ReLU 函数重新训练，仍存在 Loss 较大且反复震荡、拟合不到位的问题。
2. 数据分布关键优化：观察到现实中新冠疫情的确诊人数在当时剧增，推测也许是呈现指数爆炸增长，普通线性回归网络难以适配，遂考虑改用指数模型；考虑到指数的爆炸增长，对标签y做对数变换，把指数增长趋势映射为近似线性趋势。再对对数后标签做标准化，进一步缩小数值分布、加速网络收敛。通过该方法，彻底解决原始数据非线性过强、模型拟合困难的核心问题。
3. 优化器选型迭代：最初使用 RMSprop 优化器，训练过程 Loss 震荡明显、收敛不稳定，更换为 Adam 优化器（结合动量 Momentum + 自适应学习率）收敛更平滑、梯度下降更稳定，有效缓解训练震荡问题。
4. 过拟合问题的解决：多次训练发现模型在训练集误差很低、验证集误差偏高，存在严重过拟合。在多次调试后观察到在三层神经网络中Dropout(0.5)有较好的过拟合抑制效果，在每一层全连接后加入 Dropout(0.5) 随机失活强制网络学习通用特征，泛化能力得到了显著的提升，验证集的Loss大幅下降。
5. 动态学习率调度：最初固定的Learing Rate会导致Loss在一个较大的区间内来回震荡，通过前面大量调试还是存在相似问题，于是考虑在学习过程中逐步递减学习步长，引入了ReduceLROnPlateau。经多次调试确定该方法能是模型前期快速收敛、后期精细微调，最终Loss在较低水平保持稳定。
